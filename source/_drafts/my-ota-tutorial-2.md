本文实现前端注册与登录界面。

#### 创建前端项目
利用目前最流行的前端脚手架，创建名为ota_fe的项目。
```bash
create-react-appp ota_fe
```

#### 安装antd
我打算用目前比较流行的ui框架，来构建前端界面。
当然还有比不可少的react-redux, react-router。
```
yarn add antd react-redux react-router react-router-dom
```
